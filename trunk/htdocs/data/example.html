Example

In the example below you see the result of the 15 Kb POBS program that's been processed by POBS itself. Both variables, functions and constants have been replaced. Also, comments and indents have been removed and lines have been concatenated.

For sucessfully POBSing POBS you only need to add the variable "PA" to $UdExcVarArray. PA is used in the query_string of POBS.

The words you can still recognize are:

  • Standard PHP functions
  • Excluded Variables like $params
  • HTML code
  • Fixed text
  • Filenames
